Sequences and Series

The following applet can be used to graph a sequence, an, and the corresponding series (i.e., the sequence of partial sums sn = a1 + a2 + ... + an). Simply enter an, the function that gives the nth term of the sequence. Keep in mind that the applet plots an for n≥ 1, so be sure that an is defined for all positive integers n.

The user can also choose between Comparison Test, Limit Comparison Test, Ratio Test and Root Test to help determine the convergence of the original series.

Note that for large values of x-scale, the applet does not plot every element of the sequence. To force the applet to plot every element of the sequence, hold down the "SHIFT" key. Keep in mind that doing so may slow down the applet.
