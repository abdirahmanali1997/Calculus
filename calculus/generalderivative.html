The General Derivative of a Function

The following applet can be used to approximate the derivative of f(x) by drawing the graph of the difference quotient

for various values of h, where z is a function of x and h. Simply enter the functions f(x), z(x,h) and the value h. The applet automatically draws the graph of f(x) and the corresponding difference quotient. As z(x,h) approaches x, the graph of the difference quotient will approximate the graph of the derivative of f(x).

The value of h can be changed by simply typing a new value, such as "1.2345", "pi/2", "sqrt(5)+cos(3)", etc. You may also change this value by using the up/down arrow keys. To move the center of the graph, simply drag any point to a new location. To label the x-axis in radians (i.e. multiples of pi), click on the graph and press "control-r". To switch back, simply press "control-r" again.

Here is a list of functions that can be used with this applet.

Use the above applet to determine the derivative of:

  • z(x,h) = x h
  • z(x,h) = x (sin(h) + 1)
  • z(x,h) = cos(x) sin(h) + x
  • f(x) = ax for a = 1,2,3,4,....
  • f(x) = x^n for n = 1,2,3,4,....
  • f(x) = sin(ax) for a = 1,2,3,4,....
  • f(x) = sin(ax) for a = 1,2,3,4,....
  • f(x) = cos(ax) for a = 1,2,3,4,....
  • f(x) = a^x for various values of a
  • f(x) = ln(x)
